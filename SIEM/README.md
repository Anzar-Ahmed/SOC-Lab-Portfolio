## 📌 Overview

This project demonstrates **SIEM-based log monitoring and threat detection** using Splunk. It focuses on collecting, analyzing, and detecting suspicious activities from system logs in a controlled SOC lab environment.

## 🎯 Objectives

* Monitor Windows Security Logs
* Detect brute-force and suspicious login attempts
* Analyze events using SPL (Search Processing Language)
* Build real-world SOC detection use cases

## 🛠️ Tools & Technologies

* Splunk Enterprise
* Windows 10/11 (Victim Machine)
* Kali Linux (Attacker Machine)
* Sysmon (for detailed logging)
