# 🛡️ Endpoint Security Monitoring Lab

## 📖 Overview
This lab focuses on monitoring endpoint activity using Sysmon and Windows Event Logs to detect suspicious behavior.

## 🎯 Objectives
- Monitor processes and system activity
- Detect suspicious commands and behavior
- Analyze logs for threats

## 🧰 Tools Used
- Sysmon
- Windows Event Viewer
- (Optional) Wazuh / LimaCharlie

## 🔍 Investigation Steps
1. Installed and configured Sysmon
2. Collected logs from Event Viewer
3. Identified suspicious process activity
4. Analyzed logs for indicators of compromise
